# Геометрические фигуры

Это проект на C++ для работы с различными геометрическими фигурами.

## Функциональность

Rectangle: 
          1. Расчет периметра 
          2. Расчет площади 
          3. Расчет длины диагонали
Triangle: 
          1. Расчет пермитера
          2. Расчет площади по формуле Герона
          3. Проверка на равнобедренность
